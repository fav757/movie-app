## How to use

You should have Git and Node.js installed on your device.

Develop using:

```sh
$ git clone https://github.com/fav757/movie-app
$ npm init
$ npm start
```

Create build:

```sh
$ npm run test
$ npm run build
$ cd build
$ http-server
```
